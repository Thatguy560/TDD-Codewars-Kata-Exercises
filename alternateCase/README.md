## What Century Is It Kata

### Requirements 

upper. E.g: Hello World -> hELLO wORLD

### Test Cases

```
alternateCase("abc") # => "ABC"
alternateCase("ABC") # => "abc"
alternateCase("Hello World") # => "hELLO wORLD"
```

### Input/Output Table

| Input         | Output        |
| :------------ | :------------ |
| "abc"         | "20th"        |
| "ABC"         | "21st"        |
| "Hello World" | "hELLO wORLD" |
