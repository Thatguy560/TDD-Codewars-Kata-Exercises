## Round by 0.5 steps Kata

### Requirements 

Round any given number to the closest 0.5 step

I.E.

### Test Cases

```JavaScript
solution(4.2) = 4
solution(4.3) = 4.5
solution(4.6) = 4.5
solution(4.8) = 5
```

Round up if number is as close to previous and next 0.5 steps.

```JavaScript
solution(4.75) == 5
```

### Input/Output Table

| Input                         | Output |
| :---------------------------- | :----- |
|   4.2                         | 4      |
|   4.3                         | 4.5    |
|   5.8                         | 6      |
