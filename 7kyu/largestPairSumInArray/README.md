## Largest pair sum in array

### Requirements 

Given a sequence of numbers, find the largest pair sum in the sequence.

### Test Cases

```JavaScript
largestPairSum([10, 14, 2, 23, 19]) -->  42 (= 23 + 19)
largestPairSum([99, 2, 2, 23, 19])  --> 122 (= 99 + 23)
```

### Input/Output Table

| Input              | Output                          |
| :----------------  | :-----------------              |
| [10,14,2,23,19]            | 42             |
| [-100,-29,-24,-19,19] |  0            |
